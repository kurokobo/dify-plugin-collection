## Privacy Policy

This plugin is designed to safely handle audio/video files for transcription with speaker diarization.
Protecting user privacy is our top priority.

### Types of Data Collected

This plugin collects only the following data:

- Audio/video files provided by users for transcription
- Provider credentials (API key, API base URL, model/deployment name) configured for the selected service

No other information (such as name, email address, etc.) is collected.

### How We Use Your Data

This plugin uses the data provided by users exclusively for the following purposes:

- To transcribe audio/video files with speaker diarization using the selected OpenAI or Azure OpenAI service
- To authenticate requests to the selected service using the provided credentials

Your data will not be used for any other purposes (such as training, profiling, or marketing).

### Data Sharing and Disclosure

Audio content and credentials are transmitted only to the chosen OpenAI or Azure OpenAI service to perform the requested transcription.
User data and personal information are never shared with any other third parties or external services.

### Use of Third-party Services

This plugin relies on external transcription services:

- **OpenAI**: When using OpenAI service, your audio files are processed by OpenAI's API. Please refer to [OpenAI's privacy policy](https://openai.com/privacy/) for details.
- **Azure OpenAI**: When using Azure OpenAI service, your audio files are processed by Microsoft's Azure OpenAI service. Please refer to [Microsoft's privacy documentation](https://privacy.microsoft.com/) for details.

### Data Retention and Management

This plugin does not request storage permissions.
Therefore, except for data managed by Dify itself or the external transcription service, the plugin does not independently store any files or data provided by the user on the local file system or any external servers.

### Your Rights

You may remove the authorization or uninstall this plugin at any time.

### Policy Updates

This policy may be updated from time to time due to feature additions or legal requirements.
Please refer to the latest policy in the repository for updates.
